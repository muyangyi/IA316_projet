# My notes

## Description
给出一系列未排序的学生信息（学号和成绩），输出排序后的结果    

## Input Format
每个测试样例的第一行是序列的长度n, 1 <= n <= 10000     
在接下来的n行，每行有两个int，t（学号）和 g（成绩）     
它们满足 t <= 10000 和 g <= 1000      

## Output Format
按照学生的成绩**降序**进行排序，并输出每个学生的学号和成绩   
如果成绩相同，学号较小的应该被先输出          

## Sample Input
```
4
1 56
2 44
3 56
4 89
```

## Sample Output
```
4 89
1 56
3 56
2 44
```
