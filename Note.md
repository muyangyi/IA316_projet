# My notes

## Description
给定一个只含有 0 和 1 的数组, 找到一个最长连续子数组，它包含相同数量的 0 和 1   
输出这个最长连续子数组的长度       

## Input Format
每个测试样例的第一行是数组的长度n, n <= 20000     
第二行是n个数字，用空格隔开  

## Output Format
输出包含相同数量 0 和 1 的最长连续子数组的长度      

## Sample Input 1
```
2
1 0
```

## Sample Output 1
```
2
```
最长连续子数组是 \[1, 0\]   

## Sample Input 2
```
5
1 0 1 0 0
```

## Sample Output 2
```
4
```
最长连续子数组是 \[1, 0, 1, 0 \]
